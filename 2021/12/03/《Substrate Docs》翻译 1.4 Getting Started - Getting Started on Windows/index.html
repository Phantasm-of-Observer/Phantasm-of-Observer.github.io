Getting Started on Windows（从Windows入门）

消息

对Substrate的原生开发没有很好的支持！ 强烈建议使用 [Windows Subsystem Linux](Install WSL | Microsoft Docs) (WSL) 并按照 Ubuntu/Debian 的说明进行操作。在基于 Unix 的操作系统（如 macOS 或 Linux）上，Substrate 开发是最简单的。 Substrate tutorials（教程）和操作指南（How-to Guides）中的示例使用 Unix 风格的终端来演示如何从命令行与 Substrate 交互。

如果您决定尝试使用 Windows 计算机本地构建 Substrate，请执行以下操作：

​ 1.下载并安装“Visual Studio 构建工具”：

  • 您可以通过以下链接获取：Visual Studio: 面向软件开发人员和 Teams 的 IDE 和代码编辑器 (microsoft.com)
  • 运行安装文件：vs_buildtools.exe。
  • 确保在安装 Visual C Build Tools 时包含“Windows 10 SDK”组件。
  • 重启你的电脑。

​ 2.安装Rust:

  • Rust Book 提供了详细说明，可在 rustup.rs - The Rust toolchain installer 参考。
    • 从Install Rust - Rust Programming Language (rust-lang.org) 下载
    • 根据需要运行安装文件：rustup-init.exe 用于 32 或 64 位。
      • 它不应该提示您安装 vs_buildtools，因为您在步骤 1 中已安装。
    • 选择“默认安装”。
    • 首先，您需要在 PATH 环境变量中包含 Cargo 的 bin 目录 (%USERPROFILE%.cargo\bin)。 未来的应用程序将自动拥有正确的环境，但您可能需要重新启动当前的 shell。
